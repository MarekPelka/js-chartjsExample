# js-chartjsExample

## Usage

To run in browser:
```
./run.sh
```

To run with electron:
```bash
# Clone this repository
git clone https://github.com/MarekPelka/js-chartjsExample
# Go into the repository
cd js-chartjsExample
# Install dependencies
npm install
# Run the app
npm start
```

## Package

```
npm run dist
```
